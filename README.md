## Authentication Types Covered

1. **Password-Based Authentication** - Traditional username/password systems
2. **Multi-Factor Authentication (MFA)** - Multiple verification factors
3. **Token-Based Authentication** - JWT and API tokens
4. **OAuth/OIDC** - Delegated authorization and authentication
5. **Single Sign-On (SSO)** - Unified authentication across multiple apps
6. **Certificate-Based/mTLS** - Public key infrastructure authentication
7. **Biometric Authentication** - Fingerprint, face, and iris recognition
8. **Passwordless Authentication** - Passkeys and magic links
9. **Adaptive Authentication** - Risk-based authentication
10. **Hardware Security Keys** - FIDO2 and physical tokens
11. **Device/Machine Authentication** - IoT and service authentication
12. **Session-Based Authentication** - Cookie-based web sessions
13. **HTTP Authentication** - Basic and Digest authentication
14. **Behavioral Authentication** - Continuous user verification
